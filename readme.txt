# Plugin Gutenberg Client Block

Plugin de WordPress con bloque Gutenberg para mostrar una lista de clientes con nombre, apellido y Provincia.


## Instalación

#### 1) Dirigirse a Plugins > add new plugin. Clickear en "add new plugin"

![App Screenshot](https://www.paulascalzo.com/IMG/01.jpg)

#### 2) Posteriormente clickear en "upload plugin" y cargar el zip del plugin.

![App Screenshot](https://www.paulascalzo.com/IMG/02.jpg)

#### 3) Clickear "install now".

![App Screenshot](https://www.paulascalzo.com/IMG/03.jpg)

#### 4) Clickear "activate plugin".

![App Screenshot](https://www.paulascalzo.com/IMG/04.jpg)

## Visualización en la interfaz del admin de WordPress

#### 1) Aparecerá un menú llamado "clients" con un submenú para observar una lista de todos los clientes o cargar uno nuevo.

![App Screenshot](https://www.paulascalzo.com/IMG/05.jpg)

#### 2) Así se ve la lista de los clientes (en este momento sin clientes cargados).

![App Screenshot](https://www.paulascalzo.com/IMG/06.jpg)

#### 3) Así se visualiza el "add new".

![App Screenshot](https://www.paulascalzo.com/IMG/07.jpg)

## Implementación del Bloque Gutenberg

#### 1) Una vez cargados todos los clientes procedemos a crear una página nueva y seleccionamos el bloque "client block".

![App Screenshot](https://www.paulascalzo.com/IMG/08.jpg)

#### 2) Podemos configurar cuantos clientes queremos mostrar y el orden de los mismos.

![App Screenshot](https://www.paulascalzo.com/IMG/09.jpg)

#### 3) Al visualizar la página se verá una tabla con los datos de los clientes tal cual lo especificado en la configuración del bloque. Cada fila puede ser clickeada y se puede observar el cliente en detalle. Sin embargo puede que no se visualice correctamente según como se hayan configurado los enlaces permanentes. Para eso recomiendo dirigirse a Settings > Permalinks:
